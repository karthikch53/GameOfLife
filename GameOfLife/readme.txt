Input
-----
1)When you run the application, it will prompt you for a file path.
2)Please enter the file path which has the input seed pattern
3)Valid characters are X,x and -

Rules
-----
Rules, that are required to play the game are read from a property file. golrules.properties
To add a new rule, add an entry to the property file with an implementation for the rule present in the classpath.
